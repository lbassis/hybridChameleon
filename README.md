This repository contains the scripts used on the development of my Master's thesis "Chameleon as an HPC Benchmark Tool: On the Subject of Hybrid Processors". The files are organized in three big categories:
1 - SLURM scripts: the scripts that were run on the PCAD computers to generate the data
2 - Parsing scripts: the scripts used to extract the relevant information from the executions' outputs
3 - Drawing scripts: the scripts used to draw the graphics from the parsed data
