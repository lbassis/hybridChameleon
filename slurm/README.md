The SLURM scripts available here were written for the PCAD cluster, available at UFRGS. Running it requires you to either create the output and error directories set on the SBATCH directives or
change them to yours.

If you want to execute it on the PCAD, make sure to add to the PATH environment variable the path to your StarPU/Chameleon/pajeng installation, since the current code is based on mine.
If you want to execute it elsewhere, you also need to adjust the node names on the SBATCH directives.

The beginning of the script disables the turboboost and fixes the CPUs' frequency to their maximum value. After that, the operations are ran on an order generated by shuffling the selected inputs.
